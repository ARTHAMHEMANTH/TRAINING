css flex:
**********
moving element with respect to the main axis is called css flex.
css flex is one dimensional.
bt default css flex is responsive.

css fles propreties:
*********************

1) display:flex;

it is used to convert to block level html elements to the in line elements
2) gfap:20px;
it is used to provide gap between html elements.

3)justify :content;
    i)flex:start;flex:end/center/spance-between/space-around/space-evenly;
it is usde to move the html elements/flex in horizontal dimension.
default value is flex:start

4)aligin:items;(flex:start/flex:end;/center/stretch)
it is used to move the html elements vertically which is (cross:axis)
default value is stretch
to work with aligin items height property is mandatory

5)order:
it is used to change the order of the html elements
default value is "0"
ex: order:1;

6)flex:direction;(row/row-reverse/column/column-reverse)
it is used to change the direction of html elements or flex items.
default value is row.

7)flex:wrap;(wrap/nowrap/wrap-reverse)
it is used to wrap the html elements.
defaul value is nowrap.

8)flex:grow;
it is used to specify the grow proportion to the particular item
ex: flex grow:2;

9)flex:string;
it is used to specify the collapse/compression proportion to the perticular element.
ex: flex shrink:column 2;

10)flex:basis;
it is used to specify the width of the html elements.
ex: flex-basis:30%;
